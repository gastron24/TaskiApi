# Task Management API

Тестовое задание на позицию .NET-разработчика.

##  Реализовано
- REST API для управления задачами (CRUD)
- Фильтрация по `isCompleted`
- Сортировка по `createdAt` (asc/desc)
- Валидация входных данных (FluentValidation)
- Архитектура: **Vertical Slice** (feature-based)
- Используемые технологии: .NET 8, ASP.NET Core, EF Core, PostgreSQL, MediatR, AutoMapper, Swagger
- Асинхронные операции с БД
- Unit-тест для `CreateTaskCommandHandler`
- Поддержка запуска через Docker
